# MedTech:
Создание модели прогнозирования заболеваемости населения (Калининградская область)

### Постановщик задачи
Медицинский информационно-аналитический центр Калининградской области

Задача: создать модель, которая способна прогнозировать количество пациентов по каждому виду заболевания согласно международной классификации болезней, в привязке к населенному пункту, месяцу и половозрастным характеристикам на основе многолетних данных. (Метрика - R2)
